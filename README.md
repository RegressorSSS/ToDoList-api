# To Do list

1. Создавать задачи
2. Отмечать задачи выполненными
3. Отмечать задачи невыполненными
4. Удалять задачи
5. Редактировать задачи
6. Просматривать задачи

### База данных:
Таблица tasks:
- title         `text`
- description   `text`
- is_completed  `boolean`
- created_at    `timestamp`
